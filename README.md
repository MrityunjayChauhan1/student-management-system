# Student Management System

This is a Full Stack project developed using Spring Boot and JavaScript.

## Technologies Used
- Java
- Spring Boot
- HTML
- CSS
- JavaScript
- REST API
- MySQL / Oracle Database

## Features
- Add Student
- View Student
- Update Student
- Delete Student
- Dynamic Table Display

## Project Description
This application allows users to manage student records. 
Users can add new students, view existing students, update details, 
and delete records using a simple web interface.
